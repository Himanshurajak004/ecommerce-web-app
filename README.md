# 🛒 E-Commerce Web Application

A full-stack E-Commerce web application built using **React.js, Node.js, Express.js, and MySQL**. This project allows users to browse products, add them to cart, manage cart items, and perform checkout.

## 🚀 Features

- Product Catalog
- Add to Cart Functionality
- Cart Count in Navbar
- Shopping Cart Page
- Remove Products from Cart
- Checkout Functionality
- Product Images
- Backend APIs for Product Management
- MySQL Database Integration
- Responsive UI

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Axios
- React Router DOM
- CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL (XAMPP / phpMyAdmin)
